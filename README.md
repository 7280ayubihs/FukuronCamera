# フクロン カメラ
JEX(<http://www.jex.co.jp/>)30周年記念ゆるキャラ『フクロン』が写り込むカメラアプリです。

## ダウンロード
[Releases page](../../releases) から FukuronCamera-vX.XX.XXXXXX.apk ダンロードして、  
各端末にインストールしてください。

## 対応 Android バージョン
Android 6.0 or later
