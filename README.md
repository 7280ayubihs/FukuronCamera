# フクロン カメラ
JEX(<http://www.jex.co.jp/>)30周年記念ゆるキャラ『フクロン』が写り込むカメラアプリです。

## ダウンロード
[Releases page](../../releases) から FukuronCamera-vX.XX.apk ダンロードして、  
各端末にインストールしてください。

## 対応 Android バージョン
Android 5.0  
Android 5.1  
Android 6.0 (推奨)  

## 起動確認端末（随時、募集中）
Nexus 7 2013  
Xperia Z5 SO-01H
